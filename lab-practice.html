Part A: Find Mistakes (Identify + Fix)

In the exam, you will (1) identify the mistake in one sentence and (2) write the corrected HTML.

Sample exam questions: fix common HTML structure and accessibility mistakes
UI Element / Issue Wrong HTML (with mistake) Correct UI Element Correct HTML
Image used without alternative text 
<img src="profile.jpg">
 Image with meaningful alternative text 
<img src="profile.jpg" alt="Profile photo of the user">
Navigation not wrapped in semantic element 
<div>
  <ul>
    <li><a href="home.html">Home</a></li>
  </ul>
</div>
 Navigation using semantic <nav> 
<nav>
  <ul>
    <li><a href="home.html">Home</a></li>
  </ul>
</nav>
Table headers incorrectly marked as <td> 
<table>
  <tr>
    <td>Name</td>
    <td>Marks</td>
  </tr>
</table>
 Table with header cells 
<table>
  <thead>
    <tr>
      <th scope="col">Name</th>
      <th scope="col">Marks</th>
    </tr>
  </thead>
</table>
Multiple elements share the same id 
<input id="email">
<input id="email">
 Unique id values 
<input id="email">
<input id="confirm-email">
Button used for navigation instead of link 
<button onclick="location='about.html'">
  About
</button>
 Link used for navigation 
<a href="about.html">About</a>

Part B: True / False (HTML)

  1. T/F: The <nav> element should contain the main navigation links of a page.
  2. T/F: An <img> element is accessible even if it has no alt attribute.
  3. T/F: The id attribute value must be unique within a single HTML document.
  4. T/F: Table headers should be marked using <th> instead of <td>.
  5. T/F: A <div> provides semantic meaning to screen readers by default.

Part C: Short Answer (HCI Theory)

Answer briefly and conceptually.

  1. Which stages of Norman's Seven Stages of Action help reduce the Gulf of Evaluation, and why?
  2. In GOMS, what role do Selection Rules play when multiple methods exist?
  3. Why does KLM assume expert, error-free performance instead of novice behavior?
  4. Give one example where Norman's model is more useful than KLM, and explain why.
  5. Why are perception and interpretation stages excluded from GOMS and KLM models?
